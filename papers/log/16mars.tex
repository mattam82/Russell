\subsection*{16 mars}
Il faut faire du sous-typage dans la spécification aussi:
$f : x : \subset{n}{`N}{0 \neq} "->" \subset{n}{`N}{x <}$.

Les deux phases:
\begin{description}
\item[Inférence] on donne les types imprécis, ie: dans $x > n$, 
  $n "~>" `N$.
\item[Typage] on traverse la première dérivation de typage en ajoutant
  les coercions appropriées, par exemple:
  $`G \type n : \subset{n}{`N}{0 \neq}$,  $\type_{inf} n "~>" `N$ est
  réecrit en: 
  $`G \type \pi_{1}~n : `N$.
\end{description}

\subsubsection*{Soir!}

Formalisation des trois jugements:
\begin{description}
\item[$\typed$] Typage déclaratif, système indécidable, représentant
  exactement ce qu'on veut ajouter comme fonctionnalité.
\item[$\typei$] Version algorithmique, utilisant le dernier jugement
  pour réaliser l'adéquation avec la présentation déclarative.
\item[$\judgesubi$] ``Sous-typage'', sans obligations de preuves,
  décidable et à peu près déterministe.
\end{description}

Il faudra ensuite faire la traduction dans \Coq, avec de nouveaux
jugements réecrivant les dérivations.

Propriétés à montrer:
\begin{itemize}
\item $`G \typed t : T "=>" `E U, `G \typei t : U `^ `G \judgesubi t : U \sub T$
\item $`G \typei t : T "=>" `G \typed t : T$
\end{itemize}
